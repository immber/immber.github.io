Basics of CSS Positioning

November, 8, 2014

What are the differences between relative, absolute, and fixed positioning?

Home > Blog > CSS Positioning

When it comes to CSS Positing understanding the different positioning types and their uses can be really aggravating. This week I spent over 2 hours messing with some Divs trying to get them all in a row. So here's an attempt to break this down for others, and for my own future reference.

Fixed Positioning

In my opinion Fixed Positioning is the easiest to understand so I'll start with it first. Here's an example:

#myDIV {
          position:fixed;
          width:100px;
          height:100px;
          left:10px;
          top:100px;
        }

So what does that do? It makes the 100x100 sized div always stay 10 px in from the left and 100 px down from the top of the BROWSER WINDOW. This means that if you resize the browser or scroll up or down the div will still sit in that exact spot. Sometimes you see this used for chat popups that stay where they are on the page as you scroll down a page.

Relative Positioning

This one is a little different, think of relative positioning as a good way to move just one element around without changing any other elements. When you use relative positioning you specify how much to move it with TOP, BOTTOM, LEFT, or RIGHT. Here's what that looks like: 

 #div-1 {
           position:relative;
           top:20px;
           left:-40px;
        }

The amount that the element will move, in this case 20px from the top and 40px further to the left is going to be "relative" to where it would have normally been. So when you move something relatively you move it by just that much, without worrying about what the other elements are doing. Only this element will move, everything else will stay where you put it.

Absolute Positioning

This one can be more confusing because it affects the other elements on the page. Here's an example using Absolute positioning: 

#div-2 {
           position:absolute;
           top:0;
           right:0;
          }

When you position something absolutely, you're positioning it in that specific location. So if you use Top 0, it will be at the very top of the page. If you move something to the top of the page, it no longer takes up the space it was in, so the other elements will adjust themselves based on the new location of the element that is specified as absolute.

Hopefuly that helps clarify the Fixed, Relative and Absolute positioning!
