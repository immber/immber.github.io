Initialize Git

git init

Check status

git status

Undo Changes before commit

git checkout -- filename

Add current dir all changes/files to git

git add .

Clear the staged changes ready to commit

git reset HEAD filename or git reset HEAD^ (all files)

Commit changes to git with message

git commit -m "Added all the things"

Commit changes with verbose text

git commit -v

Forking & Cloning, create local repository Fork in GitHub to my user

git clone <URL>

Keep updated with changes from original master

remote -v

Add remote

git remote add "name"

Pull changes down from Git

git pull

Push changes up to GitHub

git push remotename branch

Pull changes from upstream master

git fetch upstream master
git merge upstream/master

Make a new branch

git branch branchcname

Switch to new branch

git checkout branchname
